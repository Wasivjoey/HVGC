# Español para Todos 🌅

A progressive web app (PWA) that teaches you to **speak, read, and write Spanish**
from absolute beginner (as young as 3) to conversational — over a structured
**5‑month curriculum** with **daily tracking** and **per‑account progress**.

## What it does

- **20 themed weeks / 5 months, 120 lessons.** Starts at toddler level (greetings,
  colors, numbers, animals) and builds to verbs, directions, shopping, travel,
  work, and free conversation.
- **Four skills per lesson:**
  - **Read** — see the Spanish word/phrase with its meaning and an example sentence.
  - **Listen** — text‑to‑speech reads every word and sentence aloud in Spanish.
  - **Speak** — tap the mic and repeat; the app checks your pronunciation with
    the browser's speech recognition (Chrome / Edge / Safari).
  - **Write** — type the Spanish, with an accent keypad: á é í ó ú ñ ¿ ¡.
  - Plus multiple‑choice quizzes for listening and reading comprehension.
- **Daily tracking:** streaks 🔥, lessons completed, minutes practiced, a 7‑day
  activity strip, and overall course progress — all saved to your account.
- **Multiple users, private accounts:** each learner registers their own
  username + password; progress is tracked separately per account.
- **Installable PWA:** works offline once loaded, installs to a phone/desktop home
  screen, with its own icon and standalone window.

## Tech

- **Backend:** Python + Flask, SQLite (stdlib `sqlite3`), password hashing via
  werkzeug (pbkdf2), signed bearer tokens via `itsdangerous` (bundled with Flask).
- **Frontend:** vanilla JS single‑page app, Web Speech API (TTS + recognition),
  service worker + web manifest for PWA/offline.
- No build step and no binary assets in git — the app icons are generated by
  `make_icons.py` (pure stdlib PNG encoder).

## Run locally

```bash
cd spanish-pwa
pip install -r requirements.txt
python make_icons.py        # generate the PWA icons
python app.py               # dev server on http://localhost:3000
```

Set `SECRET_KEY` to keep sessions valid across restarts (otherwise a random one
is generated each boot):

```bash
SECRET_KEY="some-long-random-string" python app.py
```

## Deploy to Render

This folder ships a `render.yaml` blueprint.

1. Push this repository to GitHub.
2. In Render: **New +** → **Blueprint** → select the repo.
3. Render reads `spanish-pwa/render.yaml`, provisions a free web service plus a
   1 GB persistent disk mounted at `/data` (where the SQLite database lives), and
   generates a stable `SECRET_KEY`.
4. When the build finishes, open the service URL and install it to your home
   screen.

To deploy manually instead of via Blueprint, create a **Web Service** with:

- **Root Directory:** `spanish-pwa`
- **Build Command:** `pip install -r requirements.txt && python make_icons.py`
- **Start Command:** `gunicorn app:app --bind 0.0.0.0:$PORT`
- Add a **Disk** mounted at `/data` and env var `DATA_DIR=/data` so progress
  survives deploys.

## API (all JSON)

| Method | Path | Auth | Purpose |
| --- | --- | --- | --- |
| POST | `/api/register` | — | Create an account, returns a token |
| POST | `/api/login` | — | Log in, returns a token |
| GET | `/api/me` | ✔ | Current account |
| PUT | `/api/me` | ✔ | Update learner name / level |
| GET | `/api/curriculum` | — | Full 5‑month lesson plan |
| GET | `/api/progress` | ✔ | Completed lessons, streak, activity |
| POST | `/api/progress/complete` | ✔ | Record a finished lesson |

## Notes

- Speech recognition ("Speak" practice) needs a browser that supports the Web
  Speech API — Chrome, Edge, and Safari do. Where it's unavailable the app tells
  you and lets you continue. Text‑to‑speech works nearly everywhere.
- The free Render plan sleeps on inactivity; the first request after a nap takes
  a few seconds to wake.
